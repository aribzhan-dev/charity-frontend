# 🌟 Charity Platform (Charity-Frontend)

This project is a modern web application designed to organize and manage charitable events, providing users with comprehensive information about various charity organizations.

---

## 🚀 Tech Stack

The project is built using the latest and most efficient technologies:

- **React 19** – For building the user interface.
- **Vite** – Fast build tool and development environment (with Hot Module Replacement).
- **React Router Dom (v7)** – Managing navigation and application routing.
- **Axios** – Handling data exchange with the backend API.
- **Lucide React** – A collection of modern and beautiful icons.
- **Context API** – Managing authentication and localization (State Management).

---

## ✨ Key Features

- 🏠 **Home Page**: Browse through all active charity events and organizations.
- 🔐 **Authentication**: Secure Login and User Registration system.
- 📊 **User Dashboard**: Personal profile space for tracking user activities and contributions.
- 🛠 **Admin Panel**: Management suite for events, allowing admins to add or edit charity projects.
- 📄 **Event Details**: In-depth information for each charity project with participation options.
- 🌍 **Localization**: The application currently supports Russian (RU) and Kazakh (KZ) languages.

---

## 🛠 Installation & Setup

To run the project locally on your machine, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/username/charity-frontend.git
   cd charity-frontend
   ```

2. **Install necessary dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```
   *The application usually runs at `http://localhost:5173`.*
